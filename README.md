## 1.1 Медианный фильтр
Суть данного фильтра в определении медианного значения внутри окна фильтрации. То есть все значения пикселей попадавших в окно фильтрации выписываются в массив, после чего данный массив сортируется и значение которое находится в центре массива принимается как значение пикселя выходного изображения.

## 1.2 Фильтр Гаусса
Задача фильтра Гаусса - сглаживать изображение, уменьшая шум и устраняя ненужные детали, которые могут мешать анализу. Это низкочастотный фильтр, который сглаживает неравномерные значения пикселей изображения, обрезая самые высокие значения. Каждый пиксель заменяется взвешенным средним его окружения внутри окна размера K х K, приложенного в рассматриваемом пикселе. Чем ближе соседние пиксели к текущему, тем больший "вклад” они дают в ответ.

После использования фильтра Гаусса мы не только удаляем шум, но и удаляем мелкие детали и особенности, то есть изображение делаем размытым. Но данный фильт не борется с шумом "соль и перец".

# 2. Морфологические операции
## 2.1 Эрозия
Эрозия или сужение. Данная операция представляет собой логическую операцию коньюнкции. То есть если на исходном изображении в окрестности точки, в том числе и она сама, все пиксели равны 1, то данная точка на выходном изображении будет равна 1, если нет то 0. Данная операция позволяет убрать лишние одиночные пиксели на изображении, полученные после бинаризации.

Суть бинаризации в следующем, представим, что у нас есть исходное изображение I, и мы хотим выделить, на нем все пиксели яркость которых больше
определенного порога. В результате все пиксели значение яркости которых будет больше установленного порога, будут выделены белым, а те которые меньше порога
черным.
